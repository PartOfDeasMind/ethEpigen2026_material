2. Downloading the raw sequencing reads

The raw sequencing reads used in this analysis are obtained from the ENCODE project.
The FASTQ file containing CTCF ChIP-seq reads is downloaded directly and stored locally.

options(timeout=3600)
dir.create("raw")
download.file("https://www.encodeproject.org/files/ENCFF127RRR/@@download/ENCFF127RRR.fastq.gz", "raw/ctcf.fastq.gz", mode="wb") # wb -> write binary. .gz is a compressed binary file.

3. Reads QC and trimming

Before alignment, sequencing reads are evaluated and processed using Rfastp, which performs quality control and optional trimming.
The processed reads are written to a new directory so that downstream analyses use the cleaned dataset.

dir.create("rfastp.trimmed", showWarnings = FALSE)

qc <- Rfastp::rfastp("raw/ctcf.fastq.gz", thread = 4, overrepresentationAnalysis = TRUE, outputFastq = "rfastp.trimmed/ctcf")

To evaluate the nucleotide composition across sequencing cycles, we visualize the base content before and after quality control.
